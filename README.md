# Fyrir Skagaströnd — Vefsíða

Opinber vefsíða framboðsins **Fyrir Skagaströnd** í sveitarstjórnarkosningum 2026.

🌐 [fyrirskagastrond.com](https://fyrirskagastrond.com)

## Tækni

- **React** + **Vite** — Hröð og létt vefforritun
- **Framer Motion** — Fíngerðar hreyfimyndir
- **Vanilla CSS** — Sérsniðið hönnunarkerfi
- **Netlify** — Sjálfvirk dreifing og hýsing

## Uppsetning

```bash
npm install
npm run dev
```

## Framleiðsla

```bash
npm run build
npm run preview
```
